A Pen created at CodePen.io. You can find this one at http://codepen.io/jawadrashid2011/pen/ambgYV.

Uses the following API's to get a random quote and display it on page. You can press refresh to get a random quote randomly from one of the following sites. Also, added a tweet button to tweet the quote to twitter.

http://forismatic.com/en/api/
https://market.mashape.com/andruxnet/random-famous-quotes
https://quotesondesign.com/api-v4-0/
http://quotes.stormconsultancy.co.uk/random.json
